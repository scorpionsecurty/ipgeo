# 🛰️ ipgeo :
A Python-based tool and script for tracking and analyzing the geographic locations of IP addresses
The tool allows you to obtain accurate address information such as country city ISP latitude and longitude and time zone
Designed for security educational and analytical purposes only—for legal and ethical use.
Features:

🔹 Tracks a single IP address or a list of addresses

🔹 Displays geographic information in a clear and organized manner

🔹 Supports command-line interface (CLI) operation

🔹 Exports results to CSV or TXT files

🔹 Works on Linux Termux and Windows systems

🔹 Relies on the ip-api service for location analysis
